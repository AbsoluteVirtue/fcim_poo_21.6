# Лабораторная работа 1
## Абстрактные типы данных
### Задание
Используя язык С++, необходимо создать свой тип данных и соответствующие ему операции. Тип данных создается с помощью ключевого слова class или struct. В зависимости от этого необходимо правильно использовать метки public и private для иллюстрации принципа инкапсуляции в объектно-ориентированном программировании. 

Композиция данных и функциональность получившегося абстрактного типа данных может быть произвольной.

В функции main нужно показать работоспособность всех операций, созданных для своего типа данных. Например,

    abstract_data_t obj;
    init(&obj);
    assert(obj.x == 1);
    assert(obj.y == 0);
    print(obj);
    clear(&obj);
    assert(obj.z == nullptr);

Необходимо написать подробный отчет, в котором объясняются все использованные для выполнения задания механики и свойства языка С++. 
### Дополнительное задание
Используя язык С++, необходимо взять предоставленный в дополнительном файле код на языке С и переписать его, заменив структуру на class и преобразовав все свободные функции в методы получивщегося класса. Целью задания является использование инкапсуляции для создания правильного абстрактного типа данных.
